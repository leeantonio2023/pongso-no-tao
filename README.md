# 璀璨蘭嶼 · Pongso no Tao

蘭嶼三天踏查 547 張照片的互動式地圖。

## 🔗 直接看實際網站

**👉 https://leeantonio2023.github.io/pongso-no-tao/**

> 注意：你在 github.com 看到的是程式碼倉庫，要看實際運作的網站請點上面的網址。

---

## 兩個版本

- **[⏱ 時間軸版](https://leeantonio2023.github.io/pongso-no-tao/)**
  預設首頁，可從時間軸播放三天移動路線（5/7 紅・5/8 藍・5/9 橘）

- **[📍 地點版](https://leeantonio2023.github.io/pongso-no-tao/places.html)**
  依地點瀏覽，點選任一地點看該地全部照片

兩個版本互通：頁面右上角有按鈕可以隨時切換。

## 資料來源

- 照片以 EXIF GPS 自動聚成 20 個地點群
- 39 個地標位置經 Google Maps + OpenStreetMap (Nominatim) 雙來源校正
- 部分模糊地點以視覺辨識手動覆寫（例如 玉女岩 vs 五孔洞）
- 底圖：OpenStreetMap

## 操作

- 點地圖上的彩色大圓 → 看該地點全部照片
- 點小圓點 → 跳到時間軸該位置 / 開燈箱
- 拖曳時間軸 / 按 ▶ → 看完整 3 天行程動畫
- 鍵盤：空白鍵=播放暫停、左右方向鍵=上下一張
